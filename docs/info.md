## How it works

This multiplier takes two inputs: 4-bit values A and B. The output is an 8-bit value P where P equals A times B.

## How to test

To simulate the project, run the mult_tb.v file in modelsim with the mult.v and the top files compiled in the same directory.

## External hardware

There is no external hardware required for this project. All that is required is Modelsim to verify that the multipler is working correctly.
